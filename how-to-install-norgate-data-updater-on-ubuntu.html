How to install Norgate Data Updater on Ubuntu?

I’m a happy subscriber to NorgateData’s Futures package.

However, its software called NDU only works on Windows and I only work on Ubuntu.

Tough world.

Laziness made me consider renting a Windows server that would just run NDU and rsync the extracted data to my Ubuntu server, but the server + Windows license amount to 50% of the cost of the data.

I made it a point to not be lazy and stay cheap. Thanks to Wine, I was able to make NDU work on my Ubuntu machine and, surprisingly, it went rather smoothly.

Here’s what I did:

Install Wine

Download NDU

Execute this command in the terminal:

wine "YOUR_PATH_TO/ndu.4.2.1.50.setup.exe"

Start NDU as a regular program, enter your credentials, check for updates, and boom! You’re done.
